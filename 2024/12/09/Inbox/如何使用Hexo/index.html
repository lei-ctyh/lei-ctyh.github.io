开始使用

安装

安装 Git

安装 git

安装 Node.js

安装 node

Node.js 版本限制

建议自己选择 hexo 的最新稳定版

Hexo 版本 最低版本 (Node.js 版本) 最高版本 (Node.js 版本)
7.0+ 14.0.0 latest
6.2+ 12.13.0 latest
6.0+ 12.13.0 18.5.0
5.0+ 10.13.0 12.0.0
4.1 - 4.2 8.10 10.0.0
4.0 8.6 8.10.0
3.3 - 3.9 6.9 8.0.0
3.2 - 3.3 0.12 未知
3.0 - 3.1 0.10 或 iojs 未知
0.0.1 - 2.8 0.10 未知

安装 Hexo 脚手架

使用 npm 安装 hexo，安装的时候网络可能需要点特殊要求，多试几次

创建 Github Pages 仓库

  1. 填写子集的仓库地址
  2. 设置仓库为公开
  3. 添加 readme 文件
  4. 点击右下角的创建仓库

建站

初始化博客项目

创建一个文件夹，用于存放建站后的文件，并且在其文件夹下进入 git bash

启动项目

访问 http://localhost:4000/ 能够出现 hexo 的页面就算成功

部署 Hexo 到 GitHub Pages

现在我们仅仅只是将博客在本地启动起来了，别人是是访问不了的。因此我们需要将 Hexo 上传到 GitHub 上去，使他人也能够访问
