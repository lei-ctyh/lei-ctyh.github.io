HTML 元素类型：块元素、行内元素与行内块元素汇总

#前端

概述

HTML 元素根据其显示特性主要分为三种类型：块元素（block）、行内元素（inline）和行内块元素（inline-block）。理解这些元素类型的区别对于页面布局至关重要。

分类汇总

元素类型 显示特点 常见元素 CSS 设置
块元素 独占一行，可设置宽高 div, p, h1-h6, ul, li display: block
行内元素 不独占一行，不可设置宽高 span, a, strong, em display: inline
行内块元素 不独占一行，可设置宽高 img, input, button display: inline-block

核心要点总结

  1. 块元素：
    • 独占一行，前后都是换行符
    • 可以设置宽度、高度、内外边距
    • 默认宽度为父元素的 100%
    • 可以包含其他块元素和行内元素
  2. 行内元素：
    • 不独占一行，与其他行内元素在同一行显示
    • 不能设置宽度和高度（由内容决定）
    • 水平方向的 margin 和 padding 有效，垂直方向无效
    • 一般只能包含其他行内元素和文本
  3. 行内块元素：
    • 不独占一行，但可以设置宽度和高度
    • 同时具备块元素和行内元素的特性
    • 常用于需要水平排列但需要设置尺寸的元素
