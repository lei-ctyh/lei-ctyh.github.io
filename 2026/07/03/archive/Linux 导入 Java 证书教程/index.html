十一、注意事项

  1. 导入证书时，需要使用 Java 应用实际运行的 JDK。
  2. 不要将 .key 私钥文件导入 Java 信任库。
  3. .req 是证书请求文件，不需要导入。
  4. Java 默认 cacerts 密码通常是 changeit。
  5. 导入完成后必须重启 Java 应用。
  6. 如果应用启动参数指定了自定义 truststore，需要将证书导入到自定义 truststore 中。

十二、处理完成标志

完成以下步骤后，证书导入处理完成：

  1. 证书文件已上传到 Linux 服务器。
  2. 已确认 Java 应用实际使用的 JDK。
  3. 已将证书导入对应 JDK 的 cacerts。
  4. 已验证证书别名存在。
  5. 已重启 Java 应用。
